REKURSIF FUNCTION

Rekursif adalah sebuah konsep dala pemrograman di mana sebuah fungsi dapat memanggil dirinya sendiri dalam prosesnya. 2 elemen penting dalam rekursif yaitu :

Kasus Dasar (Base Case)
Base Case adalah kondisi di mana rekursif berhenti ketika kasus dasar tercapai, fungsi rekursif biasanya mengembalikan nilai tanpa memanggil dirinya sendiri lagi, sehingga menghindari perulangan tak terbatas.

Kasus Rekursif (Recursive Case)
Kasus Rekursif adalah bagian dari fungsi di mana fungsi itu memanggil dirinya sendiri untuk menyelesaikan sub-masalah yang lebih kecil. Proses ini terus berlanjut sampai mencapai kasus dasar.

Contoh 1 :

      function tampilAngka(n) {
        if (n === 0) return;
        document.writeln(n);
        return tampilAngka(n - 1);
      }

      tampilAngka(10);

Output :

Contoh 2 :

      function faktorial(n) {
        // Basis: faktorial dari 0 adalah 1
        if (n === 0) return 1;

        // Rekursif: faktorial dari n adalah n dikalikan dengan faktorial dari n-1
        return n * faktorial(n - 1);
      }
      document.writeln('Faktorial dari 5 = ' + faktorial(5));

Output :

Contoh 3 :

      function fibonacci(n) {
        if (n <= 1) return n;
        return fibonacci(n - 1) + fibonacci(n - 2);
      }

      document.writeln('Deret Fibonacci dari 8 = ' + fibonacci(8)); // Output: 5 (deret Fibonacci: 0, 1, 1, 2, 3, 5)
