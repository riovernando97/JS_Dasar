ARRAY

Array merupakan tipe data yang digunakan untuk mendeskripsikan kumpulan elemen (nilai atau variabel), yang tiap - tiap elemennya memiliki index yang dimulai dari 0, dengan kata lain array adalah variabel yang lebih sakti karena bisa menampung nilai lebih dari 1 nilai.

Ada beberapa method pada array untuk mempermudah manipulasi pada array, yaitu :
  1. Length
    2. Method yg berfungsi untuk mengetahui berapa panjang elemen atau jumlah elemen pada array.
  2. Join
    3. Menggabungkan semua elemen dalam array menjadi satu string dengan menggunakan pemisah yang ditentukan.
  3. Push
    4. Menambahkan satu atau lebih elemen ke akhir array dan mengembalikan panjang baru dari array tersebut.
  4. Pop
    5. Menghapus elemen terakhir dari array dan mengembalikan elemen tersebut.
  5. Shift
    6. Menghapus elemen pertama dari array dan menggeser semua elemen lainnya ke indeks yang lebih rendah, kemudian mengembalikan elemen yang dihapus.
  6. Unshift
    7. Menambahkan satu atau lebih elemen ke awal array dan menggeser semua elemen lainnya ke indeks yang lebih tinggi.
  7. Slice
    8. digunakan untuk membuat salinan bagian tertentu dari array ke array baru tanpa memodifikasi array asli. Metode ini menerima dua argumen opsional: start (indeks mulai) dan end (indeks selesai, eksklusif). start: Indeks mulai, di mana elemen dengan indeks ini akan disertakan dalam salinan. end: Indeks selesai (opsional), di mana elemen dengan indeks ini tidak akan disertakan dalam salinan.
  8. Splice
    9. digunakan untuk mengubah konten array dengan menghapus atau mengganti elemen yang ada dan/atau menambahkan elemen baru ke dalam array. Metode ini menerima tiga argumen atau lebih: start: Indeks di mana penghapusan/dan atau penambahan akan dimulai. deleteCount (opsional): Jumlah elemen yang akan dihapus, dimulai dari indeks yang diberikan oleh start. Jika deleteCount tidak ditentukan, semua elemen dari start hingga akhir array akan dihapus. item1, item2, ... (opsional): Elemen yang akan ditambahkan ke array (jika ada).
  9. forEach
    10. digunakan untuk menjalankan sebuah fungsi callback sekali untuk setiap elemen dalam array. Ini tidak mengembalikan nilai apa pun (undefined). Digunakan ketika Anda hanya ingin menjalankan kode untuk setiap elemen dalam array tanpa membuat array baru atau memodifikasi array yang ada.
  10. Map
    11. digunakan untuk membuat array baru dengan hasil pemanggilan fungsi callback pada setiap elemen dalam array asli. Ini mengembalikan array baru yang berisi hasil dari fungsi callback untuk setiap elemen dalam array. Biasanya digunakan ketika Anda ingin membuat array baru dengan melakukan transformasi pada setiap elemen dalam array yang ada.
  11. Filter
    12. digunakan untuk membuat array baru yang berisi elemen-elemen dari array asli yang memenuhi kriteria tertentu yang ditentukan dalam fungsi callback. Ini mengembalikan array baru yang berisi semua elemen yang mengembalikan nilai true dari fungsi callback. Biasanya digunakan ketika Anda ingin membuat array baru yang berisi subset dari elemen-elemen dalam array asli yang memenuhi kondisi tertentu.
  12. Find
    13. digunakan untuk mencari elemen pertama dalam array yang memenuhi kriteria tertentu yang ditentukan dalam fungsi callback. Ini mengembalikan nilai dari elemen pertama yang memenuhi kriteria, atau undefined jika tidak ditemukan. Biasanya digunakan ketika Anda hanya tertarik pada elemen tunggal dalam array yang memenuhi kondisi tertentu.
