Operator Penugasan (Assigment)

x = 2, nilai 2 ditugaskan ke variabel x

Macam - macam assigment :

  • x += y sama dengan x = x + y
  • x -= y sama dengan x = x - y
  • x *= y sama dengan x = x * y
  • x /= y sama dengan x = x / y
  • x %= y sama dengan x = x mod (sisa bagi) y
  • x **= 2 sama dengan x = x pangkat 2

Operator Perbandingan

Macam - macam operator perbandingan : (akan menghasilkan nilai True dan False)

  • = = (sama dengan)
  • != (tidak sama dengan)
  • = = = (sama dengan nilai dan tipe datanya sama)
  • != = (tidak sama dengan nilai dan tipe datanya)
  • > (lebih dari)
  • < (kurang dari)
  • > = (lebih besar sama dengan)
  • < = (lebih kurang sama dengan)
