IF ELSE

If-else adalah konstruksi dalam pemrograman yang digunakan untuk mengatur aliran eksekusi program berdasarkan kondisi tertentu. Ini memungkinkan program untuk membuat keputusan dan melakukan tindakan yang berbeda tergantung pada apakah kondisi yang diberikan itu benar atau salah. Dalam sebuah if-else statement, program akan mengevaluasi suatu kondisi tertentu. Jika kondisi tersebut benar (true), maka blok kode yang terkait dengan if statement akan dieksekusi. Namun, jika kondisi tersebut salah (false), maka blok kode yang terkait dengan else statement akan dieksekusi. Penggunaan if-else sangat penting dalam mengendalikan aliran program dan memungkinkan program untuk berperilaku secara dinamis berdasarkan kondisi input atau variabel yang diberikan.

Contoh 1 :

      let nilai = prompt('Masukkan nilai = ');
      if (nilai > 80) {
        document.writeln('Lulus');
      } else {
        document.writeln('Tidak lulus');
      }
