OBJECT

sebuah objek adalah struktur data yang kompleks yang mengizinkan Anda untuk menyimpan berbagai jenis data (seperti string, angka, array, fungsi, dan bahkan objek lain) dalam satu tempat. Objek dalam JavaScript terdiri dari pasangan kunci-nilai, di mana setiap kunci adalah nama properti objek dan setiap nilai dapat berupa tipe data apa pun atau dengan kata lain "Object adalah array, yg lebih sakti". Membuat objek ada berbagai macam cara yaitu sebagai berikut :

Object Literal
Object Literal adalah cara umum untuk membuat objek dengan mendefinisikan propertinya langsung dalam kode tanpa menggunakan konstruktor. Ini memudahkan pembuatan objek dengan properti-properti yang sudah diketahui sebelumnya.

Function Declaration
Function Declaration adalah cara untuk mendefinisikan sebuah fungsi di JavaScript. Ketika sebuah fungsi dideklarasikan dengan cara ini, JavaScript secara otomatis membuat sebuah objek fungsi yang bisa digunakan untuk membuat objek baru.

Constructor
Constructor adalah sebuah fungsi khusus yang digunakan untuk membuat dan menginisialisasi objek baru. Ketika sebuah objek dibuat menggunakan constructor, constructor tersebut secara otomatis dipanggil untuk menginisialisasi properti objek.

Contoh 1 :

      let person = {
        firstName: 'Kristian',
        middleName: 'Rio',
        lastName: 'Vernando',
        age: 27,
        address: {
          street: 'Jl. Serundung Permai II',
          city: 'Nanga Pinoh',
          province: 'Kalimantan Barat',
        },
      };
      document.writeln('halo nama saya ' + person.middleName + ' saya berumur ' + person.age + 
      ' tahun, dan saya sekarang tinggal di ' + person.address['street'] + ' tepatnya di provinsi ' + person.address['province']);

        // Function Declaration
        function objectMhs(nama, nim, email, jurusan) {
          let mhs = {};
          mhs.nama = nama;
          mhs.nim = nim;
          mhs.email = email;
          mhs.jurusan = jurusan;
          return mhs;
        }
  
        let mhs1 = objectMhs('Kristian Rio Vernando', '145410169', 'riojr997@gmail.com', 'Teknik Informatika');
